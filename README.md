🚀 Билим Арена — Кванттык Билим Платформасы
Билим Арена — бул мектеп окуучулары үчүн иштелип чыккан, 100% кыргыз тилиндеги билим берүүчү жана оюн платформасы. Долбоор логиканы, реакцияны жана стратегиялык ой жүгүртүүнү өстүрүүгө багытталган заманбап оюндарды камтыйт.

✨ Өзгөчөлүктөрү
Интерактивдүү Зона (AI & Computer Vision): Веб-камера аркылуу колдун кыймылы (gestures) менен башкарылуучу инновациялык оюндар.

Түнкү жана Күндүзгү режим: Колдонуучунун каалоосуна жараша интерфейстин түсүн өзгөртүү мүмкүнчүлүгү (Dark/Light mode).

Ыкчам издөө жана фильтрация: Оюндарды категориялар (Логика, Реакция, Текст ж.б.) боюнча оңой табуу.

Сүйүктүү оюндар: Каалаган оюнду "Тандалмалар" тизмесине кошуу функциясы.

Заманбап Дизайн: Glassmorphism стилиндеги интерфейс жана жылмакай анимациялар.

🎮 Оюндардын тизмеси
Классикалык оюндар:
Сүйлөмдү түз: Сөздөрдөн туура сүйлөм курап, тил байлыгыңызды сынаңыз.

Ашыкчаны тап: Логикалык катардагы туура келбеген элементти табыңыз.

Фокус Дуэль: Реакцияңызды текшерип, элементтерди тезирээк табыңыз.

Цитаделди ээлөө: Суроолорго жооп берип, стратегиялык аймактарды басып алыңыз.

Интерактивдүү (Webcam) оюндар:
Кыргыз Бүркүтү: Колдун кыймылы менен бүркүттү башкаруу (Flappy Bird стилинде).

Ылдамдык сыноо: Виртуалдык объекттерди камера аркылуу кармоо.

Неон Сүрөт: Абада кол менен сүрөт тартуу (AI жардамы менен).

🛠 Технологиялык стек
HTML5 & CSS3: Заманбап структура жана анимациялар үчүн (Custom Properties, Flexbox, Grid).

JavaScript (Vanilla): Платформанын логикасы жана фильтрация үчүн.

MediaPipe / AI: Веб-камера аркылуу жесттерди таануу үчүн колдонулат.

Local Storage: Колдонуучунун тандаган темасын жана сүйүктүү оюндарын сактоо үчүн.

🚀 Кантип иштетүү керек?
Репозиторийди көчүрүп алыңыз:

Bash
git clone https://github.com/username/bilim-arena.git
Долбоордун папкасына кириңиз:

Bash
cd bilim-arena
index.html файлын каалаган браузерде ачыңыз.

📁 Структура
index.html — Негизги бет.

portal.css — Негизги стилдер жана дизайн.

portal.js — Логика, оюндардын тизмеси жана функциялар.

/assets — Сүрөттөр жана графикалык элементтер.

/games — Ар бир оюндун өзүнчө булак коддору.
